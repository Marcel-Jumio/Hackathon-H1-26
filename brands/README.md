# /brands — Hero Brand Profiles 🎯

**Filled by:** UK Sales + US Pacific Sales (gather assets) → UK SE (turn into JSON).
**No coding required to contribute here** — use the website (see [`../CONTRIBUTING.md`](../CONTRIBUTING.md#the-no-git-way-for-everyone-else)).

## What goes here
One `brand-profile.json` per brand we want to show off in the demo. Plus any raw assets you collect
(logo files, a screenshot of their onboarding).

## How to add a brand (no terminal needed)
1. On github.com, open this folder → **Add file → Create new file**.
2. Name it like `acme-bank.json`. Paste a copy of [`../brand-profile.sample.json`](../brand-profile.sample.json) and fill in:
   - **name**, **sourceUrl**
   - **logo** (a URL to their logo image)
   - **colors.primary** (their main brand color, as `#rrggbb`)
   - **fontFamily** (their font, if you can spot it)
3. **Propose changes → Merge.** Done.

## Pick brands that demo well
Recognizable + a clear, strong primary color + a clean logo = the ramp and theming look great.
Aim for **2–3 hero brands**.

## What "good" looks like
A profile that validates against [`../brand-profile.schema.json`](../brand-profile.schema.json). If
unsure, drop the raw assets here and ping the **UK SE** to finish the JSON.
